# Clouditera Claude Code Plugins

Clouditera 维护的 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 插件市场。

## 使用方法

### 1. 添加 Marketplace

```
/plugin marketplace add https://github.com/Clouditera/claude-plugins.git
```

### 2. 安装插件

```
/plugin install x-search
```

### 3. 配置

安装后运行配置向导：

```
/x-search:setup
```

## 可用插件

| 插件 | 说明 |
|------|------|
| [x-search](https://github.com/Clouditera/x-search) | X/Twitter 搜索与获取，支持关键词搜索、推文获取、用户资料、对话线程 |
